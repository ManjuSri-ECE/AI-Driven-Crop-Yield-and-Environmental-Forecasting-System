# AI-Driven Crop Yield and Environmental Forecasting System

An intelligent agro-climate forecasting system that combines IoT-enabled environmental monitoring with Artificial Intelligence to predict crop yield and key agricultural parameters. The system integrates sensor-based data acquisition, real-time visualization, and Temporal Fusion Transformer (TFT)-based forecasting for data-driven agricultural decision-making.

## Overview

This project is designed to forecast crop yield and environmental parameters using multivariate time-series analysis. Real-time sensor data is collected through an IoT-based hardware platform and processed using AI models to generate predictive insights and trend visualizations.

## Key Features

- Crop yield prediction using Temporal Fusion Transformer (TFT)
- Forecasting of environmental parameters
- Real-time sensor data acquisition
- Live dashboard visualization
- Time-series trend analysis
- Data-driven agricultural monitoring

## Parameters Forecasted
